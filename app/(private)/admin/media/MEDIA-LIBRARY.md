# Media Library — features & how to use it

The Media Library (`/admin/media`) is the central place to upload, organise and
reuse images, video and documents. Anything uploaded here can be inserted into
posts (via the editor's **“/Media library”** command — see the post editor
guide), so you upload a file once and reuse it everywhere.

---

## Uploading

Two ways, both land the file in the library:

1. **Upload button** — click **Upload** (top right) and pick one or more files.
2. **Drag & drop** — drag files anywhere onto the library area; it highlights to
   show it'll accept the drop.

Notes:
- **Multiple files** at once are supported.
- **Allowed types:** images, video, and documents (PDF / Word). Other types are
  rejected with a message.
- **Videos** are converted to a streamable MP4 in your browser before
  uploading, so playback works everywhere (you'll see progress while it runs).
- Files upload directly to storage; large files don't bog down the page.

## Finding things

Combine any of these — they all narrow the same list:

- **Search** — the box matches file name, alt text, and the original filename.
- **Type tabs** — **All / Images / Video / PDF / Docs**.
- **Folders** (left sidebar) — **All media**, **Unfiled**, or any folder you've
  made. On small screens the folder list collapses into a sticky **picker** at
  the top so you can switch without scrolling up.
- **Tag filter** — show only files carrying selected tags (each tag shows its
  file count).
- **Refresh** — re-pull the list (e.g., after changes elsewhere).
- **Pagination** — page through results; change page size (**20 / 40 / 60 /
  100**).

## Views

Toggle between:

- **Grid** — thumbnail cards. Hover a card for quick actions:
  **Copy URL** and **Delete**. A badge shows **“Used in N posts.”**
- **Table** — a dense list with columns: **File, Type, Alt text, Size, Folder,
  Tags, Used, Updated, Actions** — better for scanning/editing metadata.

## Organising with folders

- Create, rename, and delete folders from the **Folders** sidebar.
- **Unfiled** holds anything not in a folder; **All media** shows everything.
- Move a file into a folder from its **detail panel** (below).

## Tags

- Apply tags to a file in its detail panel — pick existing tags or **type a new
  one** to create it on save.
- Use the **tag filter** to find everything with a given tag.

## File details & editing

Click any file (card or row) to open its **detail panel**, where you can:

- See a **preview** and **copy its URL**.
- Edit metadata: **Name**, **Alt text** (important for SEO/accessibility),
  **Title**, **Credit**, **Caption**, **Description**.
- **Move** it to a folder and manage its **tags**.
- **Replace the thumbnail** (useful for video/PDF previews).
- **Delete** the file (with a confirmation).

## Usage tracking & deleting

- Each file shows **how many posts use it** (“Used in N posts”).
- A file that's **in use can't be deleted** — the Delete action is disabled
  until it's no longer referenced, so you can't accidentally break a live
  article. Unused files delete freely (with a confirm).

## Using media in posts

In the post editor, type **`/Media library`** to open this gallery and insert an
image, video, or file you've already uploaded — no re-uploading. (Dropping a new
file directly into the editor also uploads it here automatically.) See the post
editor guide for details.

---

### Tips
- Always set **Alt text** on images — it's the accessibility/SEO text and, for
  images inserted into posts, doubles as the caption/alt.
- **Reuse** existing files (`/Media library`) instead of uploading duplicates.
- Use **folders + tags** together: folders for broad grouping, tags for
  cross-cutting topics.
- Give files clear **names** so search finds them later.
